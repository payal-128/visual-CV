Resume Builder - JavaFX Application
===================================

Description:
------------
Resume Builder is a simple desktop application built using JavaFX that allows users to generate a text-based resume by filling in their personal, professional, and educational information. The app then exports the resume to a `.txt` file.

Features:
---------
- Input fields for:
  - Full Name
  - Email
  - Phone Number
  - Professional Summary
  - Job Title
  - Company Name
  - Years of Experience
  - School Name
  - Degree
  - Graduation Year
- Automatically generates and saves a well-formatted resume in a text file.

Technologies Used:
------------------
- Java
- JavaFX
- JavaFX Scene Builder (optional, for UI design)

How to Run:
-----------
1. Ensure you have Java 8 or higher installed.
2. Import the project into your favorite Java IDE (e.g., IntelliJ IDEA, Eclipse).
3. Set up JavaFX SDK in your project build path.
4. Run the `Main` class to launch the application.
5. Fill out the form and click on the "Generate Resume" button.
6. Choose the destination to save the `.txt` resume file.

File Structure:
---------------
- `Controller.java` : Main logic for handling user input and generating the resume.
- `Main.java`       : Entry point of the application (not included in the upload but assumed).
- `*.fxml`          : UI layout file (not included in the upload but referenced if using Scene Builder).
